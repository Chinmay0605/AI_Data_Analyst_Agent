# 🤖 AI Data Analyst Agent

An AI-powered Data Analysis application built with Python and Streamlit that automates the complete data analysis workflow.

## Features

- Dataset Upload
- Data Validation
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Interactive Visualizations
- AI Generated Insights
- Time Series Forecasting
- PDF Report Generation

## Technologies

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Scikit-learn
- ReportLab

## Project Structure

```
agents/
utils/
models/
app.py
requirements.txt
```

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/AI_Data_Analyst_Agent.git

cd AI_Data_Analyst_Agent

pip install -r requirements.txt

streamlit run app.py
```
