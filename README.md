# 🤖 AI Data Analyst Agent

An AI-powered Data Analysis application built with Python and Streamlit that automates the complete data analysis workflow.

## 🌐 Live Demo

🚀 **Try the App:** https://aidataanalystagent-ecqtamujokd7ysfrbwlabd.streamlit.app/

## Features

- Dataset Upload
- Data Validation
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Interactive Visualizations
- AI Generated Insights (Google Gemini)
- Time Series Forecasting
- PDF Report Generation

## Technologies

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Scikit-learn
- ReportLab
- Google Gemini API

## Project Structure

## Installation

```bash
git clone https://github.com/Chinmay0605/AI_Data_Analyst_Agent.git

cd AI_Data_Analyst_Agent

pip install -r requirements.txt

streamlit run app.py
```
